# Main Title: Ghost Slayer

**tagline:** Light up the darkness, defeat the ghosts!

## Game Essence

Players roam through a haunted village, entering houses to find hidden light switches to ward off ghosts. They must survive by temporarily stunning attacking spirits until all lights are activated.

Platform: PC (Unity)

## Formal Components

Use screenshots to illustrate your answers.

### 1. Players

* Target Audience: Ages 12+ interested in casual horror and adventure games.
* Number of Players: Single-player.
* Interaction Patterns: Player versus environment.


### 2. Goals

* Primary Goal: Assemble the complete key by solving puzzles and unlock the school gate.
* Goal Communication: Objectives are outlined in initial instructions and reinforced by hints from classmates.


### 3. Processes

* Game Start: Player receives an objective to light up houses and a quick tutorial on interactions.
* Core Gameplay: Search for light switches, avoid or stun ghosts, and light up all areas.
* Game End: Win by lighting up all houses; lose if health reaches zero.


### 4. Rules

* Stun ghosts briefly but cannot defeat them.
* Each room’s lights must be activated.


### 5. Resources

* Health: Lost when attacked, partially recoverable through items.
* Stun Power: Limited but gradually replenishes.

### 6. Conflicts

* Main Challenge: Ghost attacks while searching for switches with limited stun ability.


### 7. Boundaries

* Game Boundaries: The village limits player movement, each house containing unique challenges.


### 8. Outcomes

* Success upon activating all lights, failure if health is depleted.
